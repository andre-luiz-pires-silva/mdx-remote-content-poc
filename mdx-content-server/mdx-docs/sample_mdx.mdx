## MDX on demand
* This is a POC for [MDX on demand](https://mdxjs.com/guides/mdx-on-demand/)
* Server has an API to return the content list. Content is a JSON object with:
  - ```title```: The string that will be used to create the left menu
  - ```content```: The string with the markdown text to be rendered

## Code example

This is how we convert mdx to React component:
```js
    const code = String(await compile(content.content, {outputFormat: 'function-body'}))
    const {default: Content} = await run(code, runtime)
```
